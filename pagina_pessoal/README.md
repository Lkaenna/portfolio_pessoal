# Página Pessoal da Estudante

## 📖 Sobre o Projeto

Este projeto consiste em uma página pessoal desenvolvida em HTML e CSS, com design moderno, responsivo e inspirado em tons de roxo escuro. O objetivo é apresentar informações sobre a estudante de forma organizada, elegante e de fácil navegação.

A página foi criada para servir como um portfólio pessoal simples, permitindo destacar informações acadêmicas, interesses, habilidades e formas de contato.

Prompt usado:

Codigo para página pessoal, página unica, estilo moderno, com cores de destaques voltadas para o roxo escuro, com campos para nomes, interesses, estudos e outras informações, voltado para apresentar a estudante, em html.
---

## ✨ Funcionalidades

- Página única (Single Page)
- Layout responsivo para dispositivos móveis e computadores
- Design moderno com destaque em tons de roxo
- Seção de apresentação
- Informações pessoais

## 🚀 Como Executar

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Personalize as informações com seus próprios dados.

---

## 🎨 Personalização

Você pode alterar facilmente:

- Nome da estudante
- Foto de perfil (caso deseje adicionar)
- Objetivo profissional
- Cidade
- Formação
- Habilidades
- Interesses
- Projetos
- Links de contato (GitHub, LinkedIn, e-mail)

## 📌 Objetivo

Este projeto foi desenvolvido para demonstrar conhecimentos básicos em desenvolvimento web utilizando HTML e CSS, além de servir como uma página de apresentação pessoal.

## 📱 Responsividade

A interface adapta automaticamente seu layout para diferentes tamanhos de tela, proporcionando uma boa experiência tanto em computadores quanto em smartphones.
